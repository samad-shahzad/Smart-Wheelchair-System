# Smart Wheelchair System

An Arduino-based Smart Wheelchair designed to assist mobility through joystick and Bluetooth control.

## Features

* Joystick-based movement control
* Bluetooth connectivity
* Wireless operation
* Motorized wheelchair control

## Technologies

* Arduino
* Embedded Systems
* Bluetooth Communication
* Electronics and Sensors

## Purpose

This project was developed to explore embedded systems, hardware-software integration, and assistive technology solutions.
